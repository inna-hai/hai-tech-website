מצב רחפן למלווים במייקתון

מצב הרחפן מאפשר למלווה לעבור אוטומטית בין תלמידים בעולם Minecraft Education ולראות מה כל קבוצה בונה בזמן אמת, באמצעות צילום רחפן סביב אזור הבנייה.

העמוד הזה נבנה במיוחד גם למלווים לא טכנולוגיים — פשוט עובדים לפי השלבים, אחד אחד.

מה צריך לפני שמתחילים?

Minecraft Education פתוח, עולם פעיל עם תלמידים בפנים, וגישה ל־Code Builder בתוך המשחק.

1

שלב 1 — פתיחת Code Builder

  1. בתוך המשחק ללחוץ על C
  2. ללחוץ New Project
  3. לבחור JavaScript
2

שלב 2 — מחיקת הקוד הקיים

אחרי שהעורך נפתח, למחוק את כל מה שמופיע בו.

המטרה היא להישאר עם עורך ריק, כדי להדביק את הקוד החדש בלי תוספות מיותרות.

שלב 3 — הדבקת הקוד
להעתיק את כל הקוד הבא ולהדביק בתוך עורך ה־JavaScript
let active = false
let height = 25
let radius = 18
let viewTime = 24000

let centerX = 0
let centerY = 0
let centerZ = 0

player.onChat("watch", function () {
 active = true
 player.say("Drone started")
})

player.onChat("stop", function () {
 active = false
 player.say("Drone stopped")
})

player.onChat("time", function (num1) {
 viewTime = num1 * 1000
})

player.onChat("height", function (num1) {
 height = num1
})

player.onChat("radius", function (num1) {
 radius = num1
})

loops.forever(function () {
 if (active) {

 // קפיצה לשחקן אקראי
 player.execute("tp @s @r[type=player]")
 loops.pause(500)

 // שמירת מיקום השחקן כמרכז
 centerX = player.position().getValue(Axis.X)
 centerY = player.position().getValue(Axis.Y)
 centerZ = player.position().getValue(Axis.Z)

 // נקודה 1
 player.execute("tp @s " + (centerX + radius) + " " + (centerY + height) + " " + centerZ + " facing " + centerX + " " + centerY + " " + centerZ)
 loops.pause(viewTime / 4)

 // נקודה 2
 player.execute("tp @s " + centerX + " " + (centerY + height) + " " + (centerZ + radius) + " facing " + centerX + " " + centerY + " " + centerZ)
 loops.pause(viewTime / 4)

 // נקודה 3
 player.execute("tp @s " + (centerX - radius) + " " + (centerY + height) + " " + centerZ + " facing " + centerX + " " + centerY + " " + centerZ)
 loops.pause(viewTime / 4)

 // נקודה 4
 player.execute("tp @s " + centerX + " " + (centerY + height) + " " + (centerZ - radius) + " facing " + centerX + " " + centerY + " " + centerZ)
 loops.pause(viewTime / 4)
 }
})
4

שלב 4 — הפעלת הקוד

  1. ללחוץ על ▶ Play בתוך Code Builder
  2. לחזור למשחק
חשוב מאוד: אם לא לוחצים Play — הקוד לא יעבוד.
5

שלב 5 — הפעלת מצב הרחפן

בצ׳אט של המשחק צריך לכתוב:

watch

6

שלב 6 — עצירת מצב הרחפן

כדי לעצור את מצב הרחפן, בצ׳אט של המשחק צריך לכתוב:

stop

פקודות שימושיות

פקודה פעולה
watch מתחיל מצב רחפן
stop עוצר מצב רחפן
time 30 30 שניות לכל שחקן
height 35 משנה גובה מצלמה
radius 25 משנה מרחק צילום

איך זה עובד

המערכת בוחרת שחקן אקראי בעולם, שומרת את המיקום שלו, ואז מצלמת אותו מ־4 זוויות שונות סביב אזור הבנייה.

תקלות נפוצות

לא קורה כלום לבדוק שלחצו Play בתוך Code Builder.
אין מעבר בין שחקנים לבדוק שיש עוד שחקנים בעולם.
המצלמה רחוקה מדי להקטין את radius.
המצלמה גבוהה מדי להקטין את height.
